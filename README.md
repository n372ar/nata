 Random Task Generator

Мустафина 
 Апрель 2026  
 Python 3.8+, Tkinter (GUI)

 Описание
GUI-приложение для генерации случайных задач из предопределённого списка или пользовательских. Поддерживает фильтрацию по категориям, сохраняет историю генераций в JSON и автоматически восстанавливает её при запуске.

 Функционал
-  Случайный выбор задачи из списка
-  Фильтрация по типу: `учёба`, `спорт`, `работа`
-  Отображение истории сгенерированных задач
-  Добавление собственных задач с валидацией ввода
-  Автосохранение/загрузка истории в `history.json`

 Установка и запуск
1. Убедитесь, что установлен Python 3.8+
2. Скачать репозиторий, определить все папки из проектав одну и открыть ее через любое IDE.
